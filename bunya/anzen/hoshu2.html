全数検査と抜取検査の使い分け深掘り
🔍
INSPECTION METHOD
スパナ
軍曹
🔧 スパナ軍曹「確実に良品のみを選別する」ならどちらの検査を使うか！「破壊しないと検査できない」はどちらか！！
😰 パイプ訓練生確実に良品を選別するなら全数検査……破壊試験が必要なら抜取検査……
🔧 スパナ軍曹正解だ！「破壊試験＝全数検査」は典型的な誤りだぞ！全数破壊したら商品がなくなる！抜取検査しか選択肢がない！！
🔍 全数検査と抜取検査の使い分け
検査の種類使う場面特徴
全数検査 ・確実に良品のみを選別したい場合
・取外し困難な機器の試験
・配管の水圧試験
・防火ダンパーの機能確認 		高信頼・高コスト。1個でも見逃せない場合
抜取検査 ・破壊しないと検査できないもの
・大量生産品（電線・ヒューズ等）
・検査で商品価値がなくなるもの
・連続体（電線の引張試験等） 		コスト低・一部見逃しリスクあり
「品物を破壊しなければ検査できないものには全数検査を適用する」→ 誤り。全数を破壊すると製品がなくなるため抜取検査を適用する。
「多数の製品から確実に良品のみを選別する場合は抜取検査を適用する」→ 誤り。確実に良品を選別するには全数検査が必要。抜取検査では一部不良品が混入するリスクがある。

SECTION 02
計数抜取検査と計量抜取検査の違い深掘り
📏
SAMPLING TYPE
スパナ
軍曹
🔧 スパナ軍曹ダクトの板厚や寸法を測定する検査は計数か計量か！！
😰 パイプ訓練生板厚も寸法も連続量で測るから……計量抜取検査……ですか？
🔧 スパナ軍曹正解だ！「ダクトの板厚検査は計数抜取検査」は誤りだぞ！板厚は連続値（mm単位）だから計量抜取検査だ！！
📏 計数抜取検査と計量抜取検査の違い
種類対象データ適用例前提条件
計数抜取検査 不良品数・欠点数など個数・件数（不連続値） 溶接の欠陥箇所数・外観不良個数の検査 合否の判定（良品/不良品）
計量抜取検査 長さ・質量・強度など連続量（測定値） ダクトの板厚・寸法・配管肉厚の測定 ロットの特性値がほぼ正規分布とみなせることが前提
🔴 深掘りポイント：計量抜取検査の前提条件

計量抜取検査を適用するには、ロットの特性値がほぼ正規分布とみなせることが前提条件。この前提が成立しない場合は計量抜取検査の精度が低下する。

「ダクトの板厚や寸法の検査は計数抜取検査で確認する」→ 誤り。板厚・寸法は連続量（測定値）のため計量抜取検査で確認する。

SECTION 03
配管の水圧試験・気密試験の基準深掘り
💧
PRESSURE TEST
スパナ
軍曹
🔧 スパナ軍曹水圧試験と気密試験の違いは何か！気密試験に空気以外で使える気体は！！
😰 パイプ訓練生水圧試験は水で圧力をかける……気密試験は空気または不活性ガスで行う……
🔧 スパナ軍曹正解だ！冷媒配管の気密試験は窒素等の不活性ガスを使うぞ！可燃性ガスは絶対ダメだ！！
💧 主な配管試験の種類と基準
試験の種類試験圧力・方法主な適用配管
水圧試験（耐圧試験）最高使用圧力の1.5倍以上（最低0.75MPa）・保持時間60分冷温水管・給水管・蒸気管
満水試験系統内を水で満たし30kPa以上で漏水確認排水管（隠蔽前・埋設前）
気密試験空気または不活性ガス（窒素等）で加圧・石けん液等で漏洩確認冷媒配管・ガス配管・LPG配管
通水・通気試験実際に通水・通気して系統の流れを確認給排水管・通気管
⏱️水圧試験の実施タイミング：配管途中・隠蔽前・埋戻し前、または配管完了後の被覆施工前に区間ごとに実施する（完成後では手遅れ）。
「冷媒配管の気密試験は可燃性ガスを使用して漏洩確認を行う」→ 誤り。気密試験には空気または不活性ガス（窒素等）を使用する。可燃性ガスは爆発の危険があり使用禁止。

ダクト・保温の施工検査ポイント深掘り
🌬️
🌬️ ダクト・保温の主な施工検査項目
項目確認内容基準値・注意点
ダクトの板厚設計図書の仕様と一致しているか計量抜取検査で確認（連続量）
ダクトの吊り間隔横走りダクトの吊りボルト間隔アングルフランジ：3,000mm以下、スパイラル：3,000mm以下
フランジ接合部ガスケットの均等圧縮・ボルトの均等締め片締め禁止
防火ダンパー温度ヒューズの作動温度・羽根の開閉一般系統72℃・排煙系統280℃
保温材の継目継目の方向・テープ巻き継目は千鳥（互い違い）に。同一直線上は熱橋が生じるため不可
防火ダンパーの全数確認作動試験全数が対象（抜取検査では不可）
「電線等の連続体や防火ダンパー用ヒューズの作動試験は全数検査で確認する」→ 誤り。電線等の連続体や防火ダンパーのヒューズを全数作動させると使用不能になるため抜取検査で確認する。（防火ダンパーの「作動試験」は別途。ヒューズの「破壊試験」が抜取。）

機器の試運転調整の確認項目深掘り
⚙️
COMMISSIONING
スパナ
軍曹
🔧 スパナ軍曹ポンプの試運転でベアリング温度は何℃以上上昇したら異常か！！
😰 パイプ訓練生周囲温度より40℃以上高くなっていたら異常……ですか？
🔧 スパナ軍曹正解だ！「60℃以上」「30℃以上」は引っかけ値だぞ！周囲温度＋40℃以内が正常範囲だ！！
⚙️ 試運転調整の主な確認項目と基準値
機器確認項目基準値・注意点
ポンプ軸受け部の温度上昇周囲空気温度より40℃以上高くならないこと
冷凍機インターロックの確認冷水ポンプ・冷却水ポンプ・冷却塔とのインターロックを確認してから起動
ボイラー（蒸気）低水位遮断の確認水位検出器の水位を下げてバーナー停止・警報を確認（上げる×）
冷却塔冷却水の運転水位散水状態・ボールタップの作動確認
排水用水中ポンプ自動交互運転水位上昇による自動起動・交互運転を確認（満水警報では×）
冷凍機（搬入時）気密保持の確認真空または窒素加圧で搬入された場合、気密が保持されていることを確認してから据付
「ボイラーの低水位燃焼遮断装置の確認は、水位検出器の水位を上げてバーナーが停止することを確認する」→ 誤り。水位を下げて低水位状態にし、バーナーが停止することを確認する。

ISO 9000ファミリー規格の仕組み深掘り
🌐
ISO SYSTEM
スパナ
軍曹
🔧 スパナ軍曹ISO 9001の認証は第何者機関が行うか！また要求事項の「すべて」を守ることが必須か！！
😰 パイプ訓練生認証は第三者機関（審査登録機関）が行う……要求事項はすべて守る必要はなく組織の状況に応じて除外できる……
🔧 スパナ軍曹正解だ！「すべての要求事項を必ず守る」という問題文は誤りだぞ！除外規定があることを忘れるな！！
🌐 ISO 9000ファミリー規格の詳細
🏢ISO 9001：品質マネジメントシステムの要求事項を規定した規格。あらゆる組織に適用可能。
✅認証の仕組み：企業の品質システムが要求事項に照らして妥当かを第三者機関（審査登録機関）がチェックして認証。
📋要求事項の適用：すべての要求事項が必須ではない。組織の規模・業種・状況によって除外できる項目がある（除外規定）。
🔄継続的改善：ISO 9001はPDCAサイクルを基盤として継続的な品質改善を求める。一度取得して終わりではない。
「ISO規格の要求事項はすべての事項を必ず守るものとして規定されている」→ 誤り。組織の状況によって除外できる項目があり、すべてが必須ではない。

SECTION 07
QC工程図と品質計画の内容深掘り
📊
📊 QC工程図の内容と品質計画
📋QC工程図（フロー）：品質計画を具現化するためのもの。工程の流れに沿って以下を定める。
📌QC工程図に定める内容：①管理項目（何を管理するか）②管理水準（どのレベルで管理するか）③管理方法（どうやって管理するか）④管理値を外れた場合の処置方法
🔍品質基準が明確で再現性が確保されている製品には抜取検査を適用する。
📐管工事の品質は設計図書により下限が定められる（最低限満足すべき品質基準）
「QC工程図には管理項目・管理水準・管理方法のみを定め、異常時の処置は別途定める」→ 誤り。QC工程図には管理値を外れた場合の処置方法も含めて定めておく。

品質管理の効果と費用深掘り
💰
💰 品質管理の効果と品質コスト
✅品質管理の主な効果：①品質の向上　②手直し・やり直しの減少　③工事原価の低減　④クレームの減少　⑤工期の短縮
💴品質コストの分類：予防コスト（教育・QC計画）＋評価コスト（検査・試験）＋失敗コスト（手直し・クレーム処理）
📈予防コスト・評価コストに投資すると失敗コストが大幅に減少し、総品質コストが下がる。品質管理は「コストをかける」のではなく「コストを下げる」活動。
「品質管理を徹底すると検査コストが増加するため工事原価が上昇する」→ 誤り。品質管理の徹底は手直し・やり直しの削減により工事原価の低減につながる。
